# Hall-s-plot-and-Chain-s-plot
•	Implemented Hall Plot and Chan’s Diagnostic Plot analysis using production and injection data from the Neelam–Heera offshore asset to evaluate reservoir and well performance.
	•	Generated Hall plots to monitor injectivity behavior of water injection wells and detect changes such as formation damage or fracture/channel development.
	•	Applied Chan’s plots (WOR and WOR derivative) to identify water coning and channeling mechanisms, providing insights into water production behavior in the carbonate reservoir.
